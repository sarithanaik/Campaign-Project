# Campaign Tracker Web App

A simple web application to manage marketing campaigns. Users can register, log in, add new campaigns, view all campaigns, update campaign statuses, and delete campaigns. This project uses **Flask** for the backend and **HTML/CSS/JavaScript** for the frontend.

---

## Features

1. **User Authentication**
   - User registration
   - User login

2. **Campaign Management**
   - Add new campaigns (Campaign Name, Client Name, Start Date, Status)
   - View all campaigns with a search option
   - Update campaign status (Active, Paused, Completed)
   - Delete campaigns

3. **Dashboard Summary**
   - Displays total campaigns
   - Displays active campaigns count

4. **Frontend**
   - Responsive design
   - Fixed header navigation
   - Search campaigns by name or client

---

## Technologies Used

- **Backend:** Python, Flask
- **Frontend:** HTML, CSS, JavaScript
- **Data Storage:** JSON files (`users.json`, `campaigns.json`)
- **Other:** Fetch API for frontend-backend communication
